1. **ж„Ҹеӣҫи§ҰеҸ‘пјҢзҪ®дҝЎеәҰе…ңеә•** вҖ”вҖ” з”ЁжҲ·зӣҙжҺҘиҜҙиҜқпјҢдёҚи®ҫ `/opc` еүҚзјҖпјӣдҪҺзҪ®дҝЎеәҰж—¶дё»еҠЁзЎ®и®Ө
2. **д»Јз Ғеј•ж“Һ > Prompt еј•ж“Һ** вҖ”вҖ” GateгҖҒдҫқиө–и§ЈжһҗгҖҒзҹҘиҜҶжөҒз”Ё TypeScript е®һзҺ°пјӣж„ҸеӣҫеҲҶжһҗе’ҢиҜӯд№үеҢ№й…Қз”Ё LLM
3. **иҠӮзӮ№з»„иЈ…** вҖ”вҖ” йҳ¶ж®өиҮӘдё»йҖүжӢ©иҠӮзӮ№пјҢresolver иҮӘеҠЁеӨ„зҗҶдҫқиө–е’Ңж–Үд»¶еҹҹеҶІзӘҒ
4. **Marketplace еҸӘеҲҶеҸ‘пјҢдёҚеӯҳж•°жҚ®** вҖ”вҖ” зҹҘиҜҶгҖҒи®°еҝҶгҖҒдә§еҮәзү©йғҪеңЁз”ЁжҲ·йЎ№зӣ®йҮҢ
5. **зҹҘиҜҶеұһдәҺйЎ№зӣ®** вҖ”вҖ” еҲҮжҚўзӣ®еҪ• = еҲҮжҚўзҹҘиҜҶдёҠдёӢж–Ү

8. **йҳ¶ж®өжҳҜејәзәҰжқҹ** вҖ”вҖ” entry/exit gates дёҚж»Ўи¶іеҲҷйҳ»жӯўпјҢдҪҶе…Ғи®ёеҸ—жҺ§еӣһйҖҖ
