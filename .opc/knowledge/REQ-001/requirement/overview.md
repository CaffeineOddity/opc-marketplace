# OPC Marketplace 项目概述

## 项目定位

OPC Marketplace 是一个为独立开发者（One-Person Company）设计的 Claude Code 插件市场，提供覆盖完整产品生命周期的 AI 代理和技能集合。

## 核心价值

- **一站式编排**: 通过 `/opc` 命令自动评估任务并编排合适的代理
- **全生命周期覆盖**: 产品 → 设计 → 开发 → 测试 → 部署 → 增长
- **状态持久化**: 跨会话状态管理，支持任务恢复和回滚
- **知识库**: 自动积累项目知识，确保跨阶段上下文一致性

## 架构设计

```
/opc (入口) ──→ founder-agent 评估 → 自动编排
  │
  ├── product-kit    研究/需求/头脑风暴/市场分析
  ├── design-kit     UX/UI 设计/设计系统/UI-UX-Pro-Max
  ├── dev-kit        架构/前端/后端/安全/移动/数据库/性能/MCP
  ├── qa-kit         测试规划/Bug报告/E2E测试/无障碍审计
  ├── ship-kit       部署/CI-CD/IaC/云/成本/事件响应
  ├── growth-kit     营销/数据分析/SEO
  └── docs-kit       文档/报告/演示文稿/PDF
```

## Why: 为独立开发者提供企业级的 AI 辅助能力
## How to apply: 所有功能开发应考虑插件化、可组合、跨阶段协作
